# What is MCP Handle?

**MCP Handle AI is open source MCP integration layers that let AI agents use any tool reliably at any scale. You can use our API to automate workflows across single or multiple apps with managed authentications.**

With MCP Handle API, you can equip your AI agent with any tools, without the complexity of managing authentication, handling security, or dealing with context overload.
