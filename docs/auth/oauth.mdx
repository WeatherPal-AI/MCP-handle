## What is OAuth?

OAuth (Open Authorization) is an open standard protocol that allows third-party applications to access resources on behalf of users without exposing their credentials. MCP Handle AI implements OAuth 2.0 to securely connect with services like GitHub, Slack, Gmail, Notion, and more.
