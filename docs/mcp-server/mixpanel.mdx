## Getting started

Connect to Mixpanel to track events, analyze user behavior, and generate insights from your product analytics through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install mcp_handle
          ```
          ```bash npm
          npm install mcp_handle
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a Strata MCP Server with Mixpanel">
        <CodeGroup>
          ```python Python
          from mcp_handle import MCP Handle
          from mcp_handle.types import McpServerName
          
          mcp_handle_client = MCP Handle(api_key="YOUR_API_KEY")
          
          # Create a Strata MCP server with Mixpanel
          response = mcp_handle_client.mcp_server.create_strata_server(
              servers=[McpServerName.MIXPANEL],
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { MCP HandleClient, MCP Handle } from 'mcp_handle';
          
          const mcp_handle = new MCP HandleClient({ apiKey: process.env.MCP_HANDLE_API_KEY! });
          
          // Create a Strata MCP server with Mixpanel
          const response = await mcp_handle.mcpServer.createStrataServer({
              servers: [MCP Handle.McpServerName.Mixpanel],
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.mcp-handle.dev/mcp-server/strata/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "servers": ["Mixpanel"],
              "userId": "user123"
            }'
          ```
        </CodeGroup>

        <Card title="API Reference" icon="magnifying-glass" href="/api-reference/strata/create" horizontal>
          Full Strata API endpoints
        </Card>
      </Step>
      
      <Step title="Authenticate with Service Account Credentials">
        <CodeGroup>
          ```python Python
          mcp_handle_client.mcp_server.set_strata_auth(
              strata_id=response.strata_id,
              server_name=McpServerName.MIXPANEL,
              auth_data={
                  "data": {
                      "serviceaccount_username": "YOUR_MIXPANEL_SERVICE_ACCOUNT_USERNAME",
                      "serviceaccount_secret": "YOUR_MIXPANEL_SERVICE_ACCOUNT_SECRET"
                  }
              }
          )
          ```
          
          ```typescript TypeScript
          await mcp_handle.mcpServer.setStrataAuth({
              strataId: response.strataId,
              serverName: MCP Handle.McpServerName.Mixpanel,
              authData: {
                  data: {
                      serviceaccount_username: "YOUR_MIXPANEL_SERVICE_ACCOUNT_USERNAME",
                      serviceaccount_secret: "YOUR_MIXPANEL_SERVICE_ACCOUNT_SECRET"
                  }
              }
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.mcp-handle.dev/mcp-server/strata/set-auth" \
            -H "Authorization: Bearer YOUR_MCP_HANDLE_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "strataId": "YOUR_STRATA_ID",
              "serverName": "Mixpanel",
              "authData": {
                "data": {
                  "serviceaccount_username": "YOUR_MIXPANEL_SERVICE_ACCOUNT_USERNAME",
                  "serviceaccount_secret": "YOUR_MIXPANEL_SERVICE_ACCOUNT_SECRET"
                }
              }
            }'
          ```
        </CodeGroup>
        
        <Note>
          Get your Mixpanel service account credentials from your [Mixpanel project settings](https://mixpanel.com/settings/project/). These credentials are required to access your analytics data.
        </Note>
        
        <Check>
          ðŸŽ‰ **Your Mixpanel MCP Server is ready!** Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>
      

    </Steps>
  </Tab>
  
  <Tab title="UI">
    <Steps>
      <Step title="Open Dashboard">
        Go to your [Dashboard](https://www.mcp-handle.dev/home/mcp-servers).
      </Step>
      
      <Step title="Choose Mixpanel">
        Select Mixpanel from the list of available integrations.
      </Step>
      
      <Step title="Authenticate">
        Complete the authentication flow to connect your Mixpanel service account.
      </Step>
      
      <Step title="Use in your app">
        Copy the MCP endpoint URL and add it to your MCP-supported client (Claude Desktop, Cursor, VS Code, etc.).
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Open Source">
    <Steps>
      <Step title="Clone repository (for developers to build from source)">
        ```bash
        git clone https://github.com/WeatherPal-AI/MCP-handle
        cd mcp_handle/mcp_servers/mixpanel
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/weatherpal-ai/mixpanel-mcp-server:latest
        
        # Run with Mixpanel service account credentials
        docker run -p 5000:5000 \
          -e AUTH_DATA='{"data":{"serviceaccount_username":"your_username","serviceaccount_secret":"your_secret"}}' \
          ghcr.io/weatherpal-ai/mixpanel-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "mixpanel": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
