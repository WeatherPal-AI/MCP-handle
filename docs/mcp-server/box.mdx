## Getting started

Connect to Box to manage files and collaboration through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install mcp_handle
          ```
          ```bash npm
          npm install mcp_handle
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a Strata MCP Server with Box">
        <CodeGroup>
          ```python Python
          from mcp_handle import MCP Handle
          from mcp_handle.types import McpServerName
          
          mcp_handle_client = MCP Handle(api_key="YOUR_API_KEY")
          
          # Create a Strata MCP server with Box
          response = mcp_handle_client.mcp_server.create_strata_server(
              servers=[McpServerName.BOX],
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { MCP HandleClient, MCP Handle } from 'mcp_handle';
          
          const mcp_handle = new MCP HandleClient({ apiKey: process.env.MCP_HANDLE_API_KEY! });
          
          // Create a Strata MCP server with Box
          const response = await mcp_handle.mcpServer.createStrataServer({
              servers: [MCP Handle.McpServerName.Box],
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.mcp-handle.dev/mcp-server/strata/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "servers": ["Box"],
              "userId": "user123"
            }'
          ```
        </CodeGroup>

        <Card title="API Reference" icon="magnifying-glass" href="/api-reference/strata/create" horizontal>
          Full Strata API endpoints
        </Card>
      </Step>
      
      <Step title="Authenticate">
        <CodeGroup>
          ```python Python
          import webbrowser
          
          # Open OAuth authorization page
          webbrowser.open(response.oauth_urls[McpServerName.BOX])
          ```
          
          ```typescript TypeScript
            import open from 'open';
          
          // Open OAuth authorization page
          await open(response.oauthUrls[MCP Handle.McpServerName.Box]);
          ```
          
          ```bash cURL
          # Copy and paste the OAuth URL into your browser
          echo "Visit this URL to authorize: https://api.mcp-handle.dev/oauth/box/authorize?instance_id=YOUR_INSTANCE_ID"
          ```
        </CodeGroup>
        
        <Note>
          Get your Box API key from your [Box Developer Console](https://developer.box.com/).
        </Note>
        
        <Check>
          ðŸŽ‰ **Your Box MCP Server is ready!** Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>
      
    </Steps>
  </Tab>
  
  <Tab title="UI">
    <Steps>
      <Step title="Open Dashboard">
        Go to your [Dashboard](https://www.mcp-handle.dev/home/mcp-servers).
      </Step>
      
      <Step title="Choose Box">
        Select Box from the list of available integrations.
      </Step>
      
      <Step title="Authenticate">
        Complete the OAuth flow to connect your Box account.
      </Step>
      
      <Step title="Use in your app">
        Copy the MCP endpoint URL and add it to your MCP-supported client (Claude Desktop, Cursor, VS Code, etc.).
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Open Source">
    <Steps>
      <Step title="Clone repository (for developers to build from source)">
        ```bash
        git clone https://github.com/WeatherPal-AI/MCP-handle
        cd mcp_handle/mcp_servers/box
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/weatherpal-ai/box-mcp-server:latest
        
        # Run with OAuth support (requires MCP Handle API key)
        docker run -p 5000:5000 \
          -e MCP_HANDLE_API_KEY=$MCP_HANDLE_API_KEY \
          ghcr.io/weatherpal-ai/box-mcp-server:latest
        
        # Or run with manual API key
        docker run -p 5000:5000 \
          -e AUTH_DATA='{"access_token":"your_box_api_key"}' \
          ghcr.io/weatherpal-ai/box-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "box": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
