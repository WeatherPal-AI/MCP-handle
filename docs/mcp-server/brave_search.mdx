## Getting started

Connect to Brave Search to perform comprehensive web searches, including web results, images, news, and videos through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install mcp_handle
          ```
          ```bash npm
          npm install mcp_handle
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a Strata MCP Server with Brave Search">
        <CodeGroup>
          ```python Python
          from mcp_handle import MCP Handle
          from mcp_handle.types import McpServerName
          
          mcp_handle_client = MCP Handle(api_key="YOUR_API_KEY")
          
          # Create a Strata MCP server with Brave Search
          response = mcp_handle_client.mcp_server.create_strata_server(
              servers=[McpServerName.BRAVE_SEARCH],
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { MCP HandleClient, MCP Handle } from 'mcp_handle';
          
          const mcp_handle = new MCP HandleClient({ apiKey: process.env.MCP_HANDLE_API_KEY! });
          
          // Create a Strata MCP server with Brave Search
          const response = await mcp_handle.mcpServer.createStrataServer({
              servers: [MCP Handle.McpServerName.BraveSearch],
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.mcp-handle.dev/mcp-server/strata/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "servers": ["Brave Search"],
              "userId": "user123"
            }'
          ```
        </CodeGroup>

        <Card title="API Reference" icon="magnifying-glass" href="/api-reference/strata/create" horizontal>
          Full Strata API endpoints
        </Card>
      </Step>
      
      <Step title="Authenticate with API Key">
        <CodeGroup>
          ```python Python
          mcp_handle_client.mcp_server.set_strata_auth(
              strata_id=response.strata_id,
              server_name=McpServerName.BRAVE_SEARCH,
              auth_data={
                  "api_key": "YOUR_BRAVE_SEARCH_API_KEY"
              }
          )
          ```
          
          ```typescript TypeScript
          await mcp_handle.mcpServer.setStrataAuth({
              strataId: response.strataId,
              serverName: MCP Handle.McpServerName.BraveSearch,
              authData: {
                  api_key: "YOUR_BRAVE_SEARCH_API_KEY"
              }
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.mcp-handle.dev/mcp-server/strata/set-auth" \
            -H "Authorization: Bearer YOUR_MCP_HANDLE_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "strataId": "YOUR_STRATA_ID",
              "serverName": "Brave Search",
              "authData": {
                "api_key": "YOUR_BRAVE_SEARCH_API_KEY"
              }
            }'
          ```
        </CodeGroup>
        
        <Check>
          ðŸŽ‰ **Your Brave Search MCP Server is ready!** Once authentication is complete, you can use your MCP server URL with any MCP-compatible client.
        </Check>
      </Step>
      
    </Steps>
  </Tab>
  
  <Tab title="UI">
    <Steps>
      <Step title="Open Dashboard">
        Go to your [Dashboard](https://www.mcp-handle.dev/home/mcp-servers).
      </Step>
      
      <Step title="Choose Brave Search">
        Select Brave Search from the list of available integrations.
      </Step>
      
      <Step title="Configure API Key">
        Enter your Brave Search API key to enable search functionality.
      </Step>
      
      <Step title="Use in your app">
        Copy the MCP endpoint URL and add it to your MCP-supported client (Claude Desktop, Cursor, VS Code, etc.).
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Open Source">
    <Steps>
      <Step title="Clone repository (for developers to build from source)">
        ```bash
        git clone https://github.com/WeatherPal-AI/MCP-handle
        cd mcp_handle/mcp_servers/brave_search
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/weatherpal-ai/brave-search-mcp-server:latest
        
        # Run with Brave Search API key
        docker run -p 5000:5000 \
          -e AUTH_DATA='{"api_key":"your_brave_search_api_key"}' \
          ghcr.io/weatherpal-ai/brave-search-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "brave_search": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
