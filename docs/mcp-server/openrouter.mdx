## Getting started

Connect to OpenRouter to access multiple AI models, compare model performance, and manage usage across different AI providers through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install mcp_handle
          ```
          ```bash npm
          npm install mcp_handle
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a Strata MCP Server with OpenRouter">
        <CodeGroup>
          ```python Python
          from mcp_handle import MCP Handle
          from mcp_handle.types import McpServerName
          
          mcp_handle_client = MCP Handle(api_key="YOUR_API_KEY")
          
          # Create a Strata MCP server with OpenRouter
          response = mcp_handle_client.mcp_server.create_strata_server(
              servers=[McpServerName.OPENROUTER],
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { MCP HandleClient, MCP Handle } from 'mcp_handle';
          
          const mcp_handle = new MCP HandleClient({ apiKey: process.env.MCP_HANDLE_API_KEY! });
          
          // Create a Strata MCP server with OpenRouter
          const response = await mcp_handle.mcpServer.createStrataServer({
              servers: [MCP Handle.McpServerName.OpenRouter],
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.mcp-handle.dev/mcp-server/strata/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "servers": ["OpenRouter"],
              "userId": "user123"
            }'
          ```
        </CodeGroup>

        <Card title="API Reference" icon="magnifying-glass" href="/api-reference/strata/create" horizontal>
          Full Strata API endpoints
        </Card>
      </Step>
      
      <Step title="Configure authentication">
        <Info>
          You can get your OpenRouter API key from the [OpenRouter website](https://openrouter.ai/keys).
        </Info>
        
        <CodeGroup>
          ```python Python
          # Set the OpenRouter API key for your instance
          response = mcp_handle_client.mcp_server.set_instance_auth(
              instance_id=openrouter_server.instance_id,
              auth_data={
                  "api_key": "YOUR_OPENROUTER_API_KEY"
              }
          )
          ```
          
          ```typescript TypeScript
          // Set the OpenRouter API key for your instance
          const response = await mcp_handle.mcpServer.setInstanceAuth({
              instanceId: openrouterServer.instanceId,
              authData: {
                  api_key: "YOUR_OPENROUTER_API_KEY"
              }
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.mcp-handle.dev/mcp-server/instance/set-auth" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "instanceId": "YOUR_INSTANCE_ID",
              "authData": {
                "api_key": "YOUR_OPENROUTER_API_KEY"
              }
            }'
          ```
        </CodeGroup>
        
        <Check>
          ðŸŽ‰ **Your OpenRouter MCP Server is ready!** You can now use your MCP server URL with any MCP-compatible client to access multiple AI models.
        </Check>
      </Step>
      

    </Steps>

  <Tab title="UI">
    <Steps>
      <Step title="Open Dashboard">
        Go to your [Dashboard](https://www.mcp-handle.dev/home/mcp-servers).
      </Step>
      
      <Step title="Choose OpenRouter">
        Select OpenRouter from the list of available integrations.
      </Step>
      
      <Step title="Configure API Key">
        Enter your OpenRouter API key to enable multi-model access.
      </Step>
      
      <Step title="Use in your app">
        Copy the MCP endpoint URL and add it to your MCP-supported client (Claude Desktop, Cursor, VS Code, etc.).
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Open Source">
    <Steps>
      <Step title="Clone repository (for developers to build from source)">
        ```bash
        git clone https://github.com/WeatherPal-AI/MCP-handle
        cd mcp_handle/mcp_servers/openrouter
        ```
      </Step>
      
      <Step title="Run with Docker">
        ```bash
        # Pull the Docker image
        docker pull ghcr.io/weatherpal-ai/openrouter-mcp-server:latest
        
        # Run with OpenRouter API key
        docker run -p 5000:5000 \
          -e AUTH_DATA='{"api_key":"your_openrouter_api_key"}' \
          ghcr.io/weatherpal-ai/openrouter-mcp-server:latest
        ```
      </Step>
      
      <Step title="Configure MCP client">
        ```json
        {
          "mcpServers": {
            "openrouter": {
              "url": "http://localhost:5000/mcp/"
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
