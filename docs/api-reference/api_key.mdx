### Personal Account
1. Go to **[Dashboard](https://www.mcp-handle.dev/home/)**
2. Navigate to **API KEY** section

### Team Account
1. Go to **[Dashboard](https://www.mcp-handle.dev/home/)**
2. Navigate to **Team**
3. Access **API KEY** section
